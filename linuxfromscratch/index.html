Bem-vindo ao Linux From Scratch!

Linux from Scratch (LFS) é um projeto que possibilita a você, através de intruções passo-a-passo, construir seu próprio sistema Linux inteiramente a partir do código-fonte.

Atualmente, o Linux From Scratch consiste nos seguinte sub-projetos:

  • LFS :: Linux From Scratch é o livro principal, a base da qual derivam todos os outros projetos.
  • BLFS :: Beyond Linux From Scratch ajuda a você extender sua instalação LFS já finalizada em um sistema mais customizado e útil.
  • ALFS :: Automated Linux From Scratch oferece ferramentas para a automação e gerenciamento de construções LFS e BLFS.
  • CLFS :: Cross Linux From Scratch fornece os meios para compilação cruzada de um sistema LFS em vários tipos de sistema.
  • Dicas :: As dicas consistem numa coleção de documentos que explicam como aprimorar seu sistema LFS através de meios não inclusos nos livros LFS e BLFS.
  • Patches :: O projeto "Patches" serve como um repositório central com todos os patches úteis para um usuário LFS.
